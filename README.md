# Iris Classifier

A simple machine learning project to classify iris flowers using a **Decision Tree Classifier** in Python.

## Description

This project uses the famous **Iris dataset** to predict the type of iris flower (Setosa, Versicolor, or Virginica) based on features like petal length, petal width, sepal length, and sepal width. It demonstrates a **supervised learning** approach in machine learning, specifically **classification**.

## Features

- Load and explore the Iris dataset
- Split the dataset into training and testing sets
- Train a Decision Tree Classifier
- Predict flower types for new data
- Evaluate model accuracy

## Technologies Used

- Python
- scikit-learn
- Jupyter Notebook or any Python IDE
